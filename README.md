# Nodejs Todo API

### Setup
Install modules:
```
npm install
```

##### Optional
Setup API documentation on index:
```
npm install -g apidoc
apidoc -i app/ -o apidoc/ -t apiTemplate/
```
sudo before `npm install -g apidoc`, if on Linux/Mac
